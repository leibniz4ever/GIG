GIG - Prerequisites

The following prerequisites are required for GIG installation.

  • Java 1.5 or later
  • Eclipse SDK version 3.7 or later
  • Eclipse C/C++ Development Tools (CDT) version 8.0.0 or later
  • Eclipse Parallel Tools Platform (PTP) version 5.0.0 or later is recommended, but currently not required.

Operating Systems and Architectures Supported
Operating System Architecture
Linux x86, x86_64
